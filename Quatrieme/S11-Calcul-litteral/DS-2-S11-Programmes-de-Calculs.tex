\begin{exercicedevoir}[1]

On considère les programmes de calcul suivants :

\medskip

\textbf{Programme A :}\\
- Choisir un nombre.\\- Le multiplier par $-6$.\\- Ajouter $8$.

\medskip

\textbf{Programme B :}\\
- Choisir un nombre.\\- Soustraire $4$.\\- Le multiplier par $-2$.\\- Soustraire $4$ fois le nombre choisi.

\medskip

\begin{enumerate}[label=\alph*)]
\item Tester ces programmes avec les nombres $-8$ et $5$.\\
\item Prouver que les deux programmes donnent les mêmes résultats quel que soit le nombre choisi.

\medskip

\end{enumerate}
\end{exercicedevoir}
